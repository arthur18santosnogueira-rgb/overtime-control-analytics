# Overtime Control Analytics

## 🎯 Objetivo
Desenvolver um sistema para monitoramento e análise de horas extras em ambiente industrial, auxiliando na tomada de decisão.

## 🌍 ODS Relacionado
ODS 8 - Trabalho Decente e Crescimento Econômico

## 💡 Problema
A falta de controle estruturado sobre horas extras dificulta a análise de custos, produtividade e bem-estar dos colaboradores.

## 🚀 Solução
Sistema web com frontend e backend para registro, análise e visualização de horas extras por meio de dashboards.

## 🛠 Tecnologias (planejadas)
- Frontend: React  
- Backend: Node.js  
- Banco de Dados: PostgreSQL  

## 📅 Metodologia
O projeto será desenvolvido utilizando Scrum, com sprints quinzenais e acompanhamento via GitHub Projects.
